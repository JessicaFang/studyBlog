    Array.from的使用

    Jessica 2022/11/17 ES6

    # Array.from：就是将一个类数组对象或者可遍历对象转换成一个真正的数组

    # 1、将类数组对象装换成真正的数组

    类数组对象：最基本的要求就是具有length属性的对象

    let arrayLike={
  0:"tom",
  1:"65",
  2:"男"
  length:4
}
let arr=Array.from(arrayLike) //["tom","65","男"]

let map=new Map();
map.set("name","wang");
Array.from(map)====>[["name","wang"]]

    let arrayLike = {
    'name': 'tom', 
    'age': '65',
    'sex': '男',
    'friends': ['jane','john','Mary'],
    length: 4
}
let arr = Array.from(arrayLike)
console.log(arr)  // [ undefined, undefined, undefined, undefined ]

    总结：将类数组结构转换成数组结构，必需具备一下条件

    1、该类数组对象必须要有length，该长度为转后的数组的长度

    2、该类数组对象的属性名必须为数值性或者字符串型的数字

    # 2、将Set结构的数据转换成真正的数组

    let arr=[1,2,3,4];
let set=new Set(arr);
console.log(Array.from(set));
//这种做法可以实现去重，因为set的特点是里面的所有元素都是唯一的，并且返回是键对值完全一样的对象，然后再通过Array.from将set转换成真正的数组，这样子就得到了去重的数组

    Array.form还可以接收函数作为第二个参数，作用类型数组中map方法，对每个元素进行处理

    let arr=[1,2,3,4];
let set=new Set(arr);
console.log(Array.from(set,item=>item+1)); //[2,3,4,5]
